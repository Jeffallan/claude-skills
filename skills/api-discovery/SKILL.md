# API Discovery Skill

This skill allows Chimera to search for and identify useful public APIs for trading, finance, and data collection. It leverages the "Public APIs" resource to find endpoints for crypto data, news, and on-chain analytics.

## Triggers
- "Find an API for..."
- "What public APIs are available for..."
- "Search for data sources..."
- "API discovery..."

## Capabilities
- Search through a curated list of public APIs.
- Filter by category (e.g., Cryptocurrencies, Finance, Economics).
- Provide documentation links and authentication requirements.

## Examples
- "Find a public API for Solana real-time prices."
- "What APIs can I use for historical Bitcoin sentiment?"
- "Discover new data sources for DeFi analytics."
