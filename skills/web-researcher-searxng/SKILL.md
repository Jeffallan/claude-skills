# Web Researcher (SearXNG) Skill

This skill empowers Chimera with deep web research capabilities using SearXNG. It allows for finding off-chain alpha, news, and complex project details that aren't captured by standard market scan tools.

## Triggers
- "Search the web for..."
- "What is the latest news on..."
- "Find project details for..."
- "Deep web research..."
- "Google search for..." (aliased to SearXNG)

## Capabilities
- Multi-engine search via SearXNG.
- Aggregated results from news, social media, and forums.
- Contextual extraction of alpha and sentiment from web articles.

## Examples
- "Search the web for the latest rumors about the Uniswap v4 launch date."
- "Find project details and whitepapers for $WLD."
- "Deep web research on the recent exploit on Base."
